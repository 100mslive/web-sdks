## VideoTile

import { Story } from '@storybook/addon-docs';

Building a `<VideoTile />` component is all about composing styled/unstyled component provided by `@100mslive/react-ui`. We will create a basic videotile by composing `StyledVideoTile`.

```jsx
import { StyledVideoTile, Video } from '@100mslive/react-ui';

const VideoTileStory = () => {
  return (
    // width,height of the tile
    <StyledVideoTile.Root css={{ width: 500, height: 300 }}>
      <StyledVideoTile.Container>
        {/* trackId of the peer */}
        <Video trackId="1" />
        {/* Meta info */}
        <StyledVideoTile.Info>Deepankar</StyledVideoTile.Info>
        <StyledVideoTile.AudioIndicator>
          <MicOffIcon />
        </StyledVideoTile.AudioIndicator>
      </StyledVideoTile.Container>
    </StyledVideoTile.Root>
  );
};
```
